# Atividade 2)
Utilize o console para realizar a compilação e execução de um programa nas seguintes linguagens:
Não é permitido utilizar o ambiente de desenvolvimento

- C     (gcc)
1. Adicionar o "C:\Program Files (x86)\Dev-Cpp\MinGW64\bin" nas variáveis de ambiente do sistema:
2. Rodar
$ gcc hello_world.c
$ hello_world.exe
Hello World

- Java  (javac)
1. Instalar JDK
2. Adicionar o caminho "javac" nas variáveis de ambiente do sistema:
3. Rodar
$ javac HelloWorld.java
$ Java HelloWorld
Hello World


- C#    (csc)
1. Adicionar o "C:\Windows\Microsoft.NET\Framework\v2.0.XX" nas variáveis de ambiente do sistema:
2. Rodar
$ csc hello_world.cs
$ hello_world.exe
Hello World
